CDL Dashboard

Student portal for tracking CDL training progress, documents, and scores

Open App in New Tab

View Source on GitHub

Overview

This project involved using Claude Code to develop a full student dashboard for The Trucking School. The CDL Dashboard provides students with a centralized portal to track their CDL training progress, view test scores, access certificates, and manage account settings.

The primary purpose of this app is to:

  • Give students real-time visibility into their training progress across CDL classes
  • Provide a clean, professional interface for viewing test scores and earned certificates
  • Streamline the student experience with a modern, responsive dashboard

The Problem

CDL training programs often lack a digital student experience. Students have limited visibility into their progress, scores, and certification status, relying on phone calls or in-person visits to get updates. This creates friction for both students and administrators.

The Solution

The CDL Dashboard addresses this gap with a purpose-built student portal featuring:

  • Dashboard Home — Welcome overview with key stats, training progress, and recent activity
  • Profile — Student information, enrollment details, and class history
  • Documents — Earned certificates with download access and in-progress tracking
  • Training — Module-by-module progress with completion status and scores
  • Testing — Complete score history with pass/fail indicators and averages
  • Settings — Notification preferences, appearance options, and account management

Technical Details

Framework/Stack: Next.js 14, React 18, TypeScript, Tailwind CSS

Deployment: Vercel (production) + GitHub (source control)

Design: Dark theme with glassmorphism cards, red accent branding, responsive sidebar layout, staggered animations

Tools Used:

  • Claude Code — AI-assisted development for building components, pages, and data models
  • Vercel — Production deployment and hosting
  • GitHub — Version control and source code management

Product Requirements Document

The CDL Dashboard was built from a detailed implementation plan that specified:

  • 6 dashboard pages with consistent navigation and design language
  • Mock data for a sample student (John Martinez) with Class A (complete, 87%) and Class B (in progress, 67%)
  • Reusable components including StatCard, ProgressBar, ScoreBadge, CertificateCard, and ActivityItem
  • Responsive design with a collapsible sidebar for mobile and a sticky header
  • Dark glassmorphism theme matching The Trucking School branding (slate-900 background, #e00000 red accent, Lexend Deca font)
